A lightweight browser extension that optimizes the reading and archiving experience for documents on Studocu.

> **Status:** v2.0

### Introduction

This tool was built to remove visual obstacles that get in the way when viewing study documents. Instead of performing complex manual tweaks, the extension provides a "one‑click" solution to clean up the interface and export documents in a print‑ready format.

### Key Features

#### 1. Bypass Blur & Remove Watermark
Core functionality that reveals the full, original content of the document:

- **Remove blur overlay (Unblur):** Strips away layers that hide the content so the text is displayed clearly at 100%.
- **Remove watermark:** Automatically hides watermarks, overlay text, ads, and disruptive popups that sit on top of the content.
- **Optimized display:** Preserves original formatting (fonts, layout) to maintain the best possible reading experience.

#### 2. PDF Export
Features that help you save documents for printing or offline reading:

- **Automatic rendering:** Automatically scrolls and loads all pages of the document before exporting.
- **Standard A4 sizing:** Automatically adjusts margins and page size to match common printers.
- **Display bug fixes:** Permanently fixes the common "black line" artifact at the bottom of pages when saving web pages as PDF.

### Installation

Because this is a personal development tool (not yet on the web store), you need to install it manually in Developer Mode:

1. **Download the source code:** Download the project `.zip` and extract it (or clone this repository).
2. **Open the extensions manager:** Go to `chrome://extensions/` in your browser (Chrome, Edge, etc.).
3. **Enable Developer Mode:** Toggle **"Developer mode"** in the top‑right corner.
4. **Load the extension:** Click **"Load unpacked"** and select the folder containing the extracted source code.

### How to Use

1. Open the Studocu document you want to view.
2. If the document is blurred or covered with watermarks, open the extension and click **"Bypass blur & watermark"** first so that later steps work correctly.
3. Scroll to the bottom of the page to make sure all content is fully loaded.
4. Open the extension and click **"Create PDF file"**.
5. Wait a few seconds for the tool to process; the browser’s "Save as PDF" dialog will appear automatically.
